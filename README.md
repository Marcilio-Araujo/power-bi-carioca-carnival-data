## üìä Project Overview

The goal of this project is to apply hands-on learning in Power BI by
developing a complete analytical report that includes:

-   Data ingestion and transformation using **Power Query**
-   Data modeling best practices, including relationships and
    star-schema concepts
-   Creation of calculated columns and measures using **DAX**
-   Design of interactive reports and dashboards
-   Implementation of slicers, filters, and navigation elements
-   Use of themes, custom visuals, and visual best practices

The exercises simulate real-world business scenarios and demonstrate how
data can be transformed into actionable insights.

-   Interactive charts and tables

-   A consistent and professional visual theme

-   Organized navigation
